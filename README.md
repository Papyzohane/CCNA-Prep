# CCNA-Prep

Ce dépôt regroupe une série de 20 exercices pratiques conçus pour consolider les acquis en administration réseau, couvrant l'ensemble du programme abordé durant mon BUT. L'objectif est de préparer la certification CCNA (Cisco Certified Network Associate) tout en révisant les configurations critiques de l'infrastructure.

🛠️ Structure des exercices

Le projet est découpé en modules progressifs, allant de la configuration de base à la sécurité avancée.

🔵 Niveau 1 : Commutation (Switching)

    Ex 01-05 : Fondamentaux (Hostname, Mots de passe, SSH).

    Ex 06-08 : Segmentation réseau (VLANs, Trunking 802.1Q, VTP).

    Ex 09-10 : Redondance et performance (Spanning Tree Protocol, EtherChannel L2/L3).

🟢 Niveau 2 : Routage (Routing)

    Ex 11-13 : Routage Inter-VLAN (Router-on-a-Stick, SVI).

    Ex 14-16 : Protocoles de routage dynamique (OSPFv2 monopoint et multipoint).

🔴 Niveau 3 : Services & Sécurité

    Ex 17-18 : Services IP (DHCP Relay, NAT Statique/Dynamique/PAT).

    Ex 19-20 : Sécurisation des accès (Standard & Extended ACLs, Port Security).

📂 Organisation des fichiers

Pour chaque exercice, tu trouveras deux types de fichiers afin de favoriser l'apprentissage :

    Version "Starter" (Non configurée) : La topologie réseau est prête, mais aucun équipement n'est paramétré. C'est le point de départ idéal pour s'exercer en autonomie.

    Version "Final" (Configurée) : La solution complète avec l'ensemble des commandes appliquées. Elle peut servir de corrigé ou de référence pour vérifier vos propres configurations
